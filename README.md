# 📝 Task no Kokyū – Todo Management Web App
![Carbon Bazaar Architecture](public/Screenshot%202026-03-11%20130540.png)

Task no Kokyū is a modern web-based task management application designed to help users organize daily activities and track their productivity.

The application provides a simple, responsive interface for adding, viewing, and managing tasks efficiently.

🌐 **Live Demo:**  https://tasknokokyu.netlify.app/

---

# 📌 Project Overview

Task no Kokyū is a lightweight productivity tool that allows users to manage their tasks in real time.

The application focuses on:

- Simple and intuitive task management
- Clean and minimal UI design
- Fast and responsive user interactions
- Efficient task organization

It demonstrates core concepts of modern frontend development such as component-based architecture and state-driven UI updates.

---

# 🚀 Core Features

- ➕ Add new tasks instantly  
- 📋 View tasks dynamically  
- 🗑️ Delete completed or unwanted tasks  
- ⚡ Real-time UI updates without page refresh  
- 📱 Fully responsive design for multiple devices  

---

# 🛠 Tech Stack

| Layer | Technology |
|------|-------------|
| Frontend | React / Next.js |
| Styling | CSS / Tailwind CSS |
| State Management | React Hooks |
| Deployment | Netlify |

---

# ⚙️ How It Works

1. Users enter a task in the input field.
2. The task is added instantly to the task list.
3. Each task appears with an option to delete.
4. Tasks can be removed once completed.

The interface updates dynamically without requiring page reloads.

# 🔧 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/x-anish-y/Todos.git
cd Todos
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

Open the application in your browser:

```
http://localhost:3000
```
